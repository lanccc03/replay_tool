Scenario View After Removing Tabs

Two approaches for the "full scenario editor"

A

New Navigation Page

"New Scenario" adds a 5th page to the left navigation (like "Scenario Editor"), replacing the dialog with a full-page editor. The existing Scenarios page becomes a read-only list. Running a scenario still navigates to Replay Monitor.

B

Replace In-Place (Stack Switch)

"New Scenario" replaces the table view with the editor view within the same Scenarios page. Clicking Cancel/Back returns to the list view. No new navigation item. This is similar to how many apps handle "create" flows inline.
